# reports/ — 대시보드 파일 가이드

- `_template.html`: 모든 대시보드의 원본. DATA SECTION만 채워 복사 → `YYYYMMDD_dashboard.html`
- `_mobile_snippet.md`: 모바일 반응형 + 전략 추천 컴포넌트 코드 블록 모음 (섹션 6 필수 포함)
- `YYYYMMDD_dashboard.html`: 날짜별 분석 결과물. `═══ END DATA ═══` 아래 코드는 절대 수정 금지
- 새 UI 기능은 `_template.html` 자체를 수정; 당일 파일은 건드리지 않음
